# Amazon App Runner

AWS App Runner is a fully managed container application service that lets you build, deploy, and run containerized web applications and API services without prior infrastructure or container experience. Start with your source code or a container image.

## APIs

- **Amazon App Runner API** - The AWS App Runner API enables programmatic management of App Runner services, including creating and managing services, auto scaling configurations, custom domains, VPC connectors, and observability configurations for containerized web applications.
